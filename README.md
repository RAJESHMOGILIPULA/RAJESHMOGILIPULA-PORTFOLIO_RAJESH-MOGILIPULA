## üì¶ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create your EmailJS account and get your Service ID, Template ID, and Public Key.
4. Update `src/components/sections/Contact.jsx` with your credentials.

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
